## PART I — THE GUIDING PRINCIPLE

The current site is a business website that contains a philosophy. Sprint 010 inverts that. It becomes a philosophy that contains a business.

The visitor does not arrive to browse products. They arrive to cross a threshold. Everything — the color of a shadow, the weight of a heading, the way a section reveals itself — must transmit the same message: *you are entering a world built on a specific way of seeing.*

That way of seeing is: **Word. Sound. Symbol. Story.** Afrofuturism not as aesthetic but as epistemology — a method of knowing and building reality through imagination, discipline, visualization, and ancestral technology.

Every page must make a non-negotiable declaration: *no one else made this. No one else could.*

## PART II — WHAT THE ART REVEALED

Before any design decision was made, the full asset library was read. Five visual laws run through every image.

### Law 1 — The Solitary Figure Against the Enormous World
Book 1: one person in ruins radiating gold rings. Book 2: one person entering the blue pool. Book 3: one person before the spiral machine. Eyes of Trust: five women in a desert canyon. The Ronin Monk: one hooded figure with a serpent. The visual philosophy is always the same — *the individual carrying enormous frequency*. Every page layout honors this. Large environments. Small figures. Vast scale.

### Law 2 — Frequency is Visible
The rings in Book 1 and Book 2 are the most important design element in the entire library. Frequency is not a concept in this universe — it is a *physical phenomenon*. It radiates outward from people in concentric circles. The website reflects this. Circular motifs, concentric rings, radial gradients centered on figures. The seal itself — flame inside a square inside a circle — is the same logic.

### Law 3 — The Real World Seen Through the Doctrine
Jamaica 1 and Jamaica 3 are the most philosophically pure images in the library. They take an actual Caribbean landscape and overlay geometric grid, dome structures, cosmic moons, teal skies. This *is* the Afrofuturist method. The site does the same — takes real elements and shows them through the lens.

### Law 4 — Four Distinct Atmospheres Already Exist
The art already codes four emotional registers:
- **Amber/Orange + Slate** — post-war city, ruins lit by frequency rings → *WORD environment*
- **Azure/Cyan + Deep Blue** — submerged ruins, blue pool, teal armor → *SOUND environment*
- **Crimson/Red + Obsidian** — obelisk desert, red sky, eye monuments → *STORY environment*
- **Teal/Gold + Deep Green** — Jamaica at night, dome structures, orange moon → *SYMBOL environment*

The four frequency pages already have their palettes. The art built the color system before design began.

### Law 5 — The Mask is the Brand
The Banners of the Fallen cover is the clearest brand image. The hooded Monk, looking upward, concentric-circle mask, teal against amber fire. The mask does not hide — it *declares*. The Ronin Monk is known by their frequency, not their face.

## PART VIII — THE VIDEO SYSTEM

**Homepage hero video:**

```html
<video autoplay muted loop playsinline
  poster="assets/img/hero/Lovejoy Frequencies Cover 1.png"
  class="threshold-video">
  <source src="assets/video/hero/resonance gate short 1.mp4" type="video/mp4">
</video>
```

Brightness reduced via CSS `filter: brightness(0.3)` on the `<video>` element. Text and seal layered above via `position: absolute; z-index: 1`.

**CSP update required in `netlify.toml`:**

Add `media-src 'self'` to the Content-Security-Policy header. One directive. No other changes.

**`prefers-reduced-motion` behavior:**
- If visitor has reduced motion enabled: video does not play, poster image shows instead
- Implemented via CSS `@media (prefers-reduced-motion: reduce) { video { display: none; } }`
- The poster image covers the zone — no broken experience

## PART XII — THE ANIMATION SYSTEM

All animations are CSS only — no libraries. All respect `prefers-reduced-motion`.

| Animation | Element | Implementation | Duration |
|-----------|---------|----------------|----------|
| **Video hero** | Threshold zone background | `<video autoplay muted loop playsinline>` | Continuous |
| **Sigil pulse** | Sky Khan Doctrine Seal on homepage | `@keyframes` scale 1→1.04→1 + drop-shadow pulse | 4s infinite |
| **Sigil ring expand** | Seal on hover | 2 rings expand outward and fade, pure CSS | 0.8s ease-out |
| **Text cascade** | Zone 1 philosophy lines | Staggered `@keyframes fadeUp` via `IntersectionObserver` | 0.5s per line, 0.2s stagger |
| **Scroll reveal** | All major sections | `data-reveal` attribute + `IntersectionObserver`, adds `.in-view` class | 0.6s ease-out |
| **Frequency card hover** | Four homepage cards | `transform: scale(1.02)` + `brightness(1.15)` + top border draws in | 0.35s ease |
| **Nav underline draw** | All nav links | CSS `::after` pseudo-element, `width: 0→100%` | 0.25s ease |
| **Character float** | Shinobi Kush on SOUND page | CSS `@keyframes` translateY(0→-8px→0) | 3s ease-in-out infinite |
| **Audio bar slide** | Fixed bottom audio strip | CSS animation-delay 5s, slides up from below viewport | 0.4s ease-out |

**The one JS file:** A single `IntersectionObserver` block in `js/main.js` adds class `in-view` to elements with `data-reveal` attribute as they enter the viewport. All animation CSS lives in `styles.css`. No library. CSP-safe.
