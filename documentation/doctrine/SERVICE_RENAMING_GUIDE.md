## IV. CTA LANGUAGE (OFFICIAL)

### Forbidden CTAs
Never use these on any page:
- "Buy Now"
- "Sign Up"
- "Get Started"
- "Learn More"
- "Download"
- "Subscribe"
- "Click Here"
- "Shop Now"
- "Book Now"

### Approved CTAs by Context

| Context | CTA Language |
|---------|-------------|
| Entering a frequency page | "Enter the [Frequency Name]" |
| Joining The Order (email) | "Join the Order" / "Transmit" |
| Acquiring an artifact | "Carry this" / "Receive this transmission" / "Acquire" |
| Starting a service | "Begin the ritual" / "Enter the engagement" / "Request the First Council" |
| Reading a blog/archive post | "Enter the transmission" / "Read" (plain — reading needs no ceremony) |
| Listening to music | "Tune in" / "Receive" |
| Contact / Council | "Open the Council Chamber" / "Send a transmission" |
| Discovery elements | "Go deeper" / "Follow the signal" / "The frequency continues here" |

### CTA Design Rules
- CTAs are never the loudest element on the page — the world is louder than the prompt
- CTAs in brass color against dark backgrounds only — never in bright contrasting buttons
- CTA text is set in Cinzel (small caps works) or Inter semibold — never Cormorant (that is reserved for doctrine)
- Hover state on CTAs: a subtle glow or underline, not a color reversal
